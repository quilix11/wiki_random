# Wiki Quiz Bot 🇺🇦

Вікторина за випадковими статтями української Вікіпедії.

## Основні функції
- 📖 **Режим «Книги»**: Пагінація довгих статей для зручного читання.
- 🧠 **AI Генерація**: Питання створюються на основі Google Gemini 2.0.
- 🛠️ **Резервний алгоритм**: Генерація питань без ШІ у разі збоїв API.
- 📊 **Статистика**: Персональна та загальна статистика гравців.
- 🏆 **Топ гравців**: Рейтинг найкращих за кількістю балів.

## Локальний запуск
1. Встановіть залежності: `pip install -r requirements.txt`
2. Налаштуйте `.env`:
   - `TELEGRAM_TOKEN`
   - `API_KEY` (Gemini)
3. Запустіть: `python telegram_bot/bot.py`

## Використання Docker
```bash
docker-compose up -d --build
```
 База даних зберігатиметься у папці `database/` на хості.
